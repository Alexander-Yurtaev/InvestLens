**Название проекта:** Portfolio Lens

**Описание:**

Приложение представляет собой платформу для мониторинга и аналитики инвестиционных портфелей, построенную на основе **микросервисной архитектуры** для обеспечения масштабируемости и надежности.

**Ключевые возможности:**

*   **Импорт и управление портфелем:** Пользователи могут загрузить структуру своего инвестиционного портфеля для дальнейшего отслеживания.
*   **Динамическая статистика и аналитика:** Визуализация ключественных метрик, включая историю стоимости портфеля, распределение активов, доходность и другие экономические параметры.
*   **Умные оповещения:** Настройка персональных уведомлений о рыночных событиях, таких как достижение ценой актива заданного уровня или значительные изменения.
*   **Продвинутый поиск по облигациям:** Мощный инструмент с гибкими фильтрами для быстрого нахождения ценных бумаг по заданным критериям (доходность, срок, кредитный рейтинг и др.).
*   **Автоматическое обновление данных:** Фоновая синхронизация с внешними финансовыми источниками гарантирует актуальность всех котировок и финансовых показателей.

**Цель проекта:** Предоставить инвесторам единый, удобный инструмент для глубокого анализа своих вложений и принятия обоснованных решений на основе актуальных данных.
